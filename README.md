# GhostBar

A WPF desktop application built with .NET 8.

## Project Structure

- `App.xaml` - Application entry point and resources
- `App.xaml.cs` - Application code-behind
- `MainWindow.xaml` - Main window UI definition
- `MainWindow.xaml.cs` - Main window code-behind
- `GhostBar.csproj` - Project file

## Requirements

- .NET 8 SDK or later
- Windows OS

## Building the Project

To build this project, you need to install the .NET SDK:

1. Download and install the .NET SDK from: https://dotnet.microsoft.com/download
2. Open a terminal in the project directory
3. Run: `dotnet build`

## Running the Application

Once the .NET SDK is installed:

```powershell
dotnet run
```

## Development

This is a WPF (Windows Presentation Foundation) application targeting .NET 8. The C# Dev Kit extension is installed in VS Code for C# development support.
